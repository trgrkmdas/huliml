# Bitcoin Long/Short Signal Prediction with LightGBM

Bitcoin fiyat hareketlerini tahmin ederek long/short sinyalleri üreten makine öğrenmesi projesi.

## Proje Yapısı

```
MLProject/
├── data/              # Ham ve işlenmiş veriler
├── src/               # Kaynak kodlar
│   ├── data_collection.py
│   ├── feature_engineering.py
│   ├── model.py
│   ├── backtesting.py
│   └── signals.py
├── notebooks/         # Jupyter notebook'lar (analiz ve denemeler)
├── models/            # Eğitilmiş modeller
├── results/           # Sonuçlar ve görselleştirmeler
├── requirements.txt
└── README.md
```

## Kurulum

```bash
pip install -r requirements.txt
```
